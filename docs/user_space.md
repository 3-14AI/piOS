# Пользовательское Пространство (User Space)

Пользовательское пространство отделено от ядра и базируется на выполнении изолированных WebAssembly (WASM) компонентов.

## Основные особенности:

1.  **WASM Runtime:**
    Используется интерпретатор (wasmi) или компилятор (Cranelift) для запуска компонентов. Среда WASM выступает как надежная "песочница", а доступ к ресурсам операционной системы предоставляется через интерфейсы WASI (WASI-Core, WASI-Crypto, WASI-NN).

2.  **Композитор Окон (WGPU Compositor):**
    Пользовательский интерфейс строится поверх оконного менеджера, использующего графический API `wgpu`. Он управляет рендерингом (через `ab_glyph` для текста) и маршрутизацией ввода-вывода.

3.  **Генеративный UI (Slint):**
    Для создания графических приложений интегрирован фреймворк Slint. Взаимодействие с AI-подсистемами позволяет генерировать пользовательский интерфейс динамически в зависимости от интентов (intents) пользователя.

4.  **Естественно-Языковая Оболочка (NL-SH):**
    Вместо классических командных оболочек (bash/sh), в системе реализован интерпретатор, переводящий текстовые запросы на естественном языке в конкретные системные вызовы ядра через эмбеддинги и inference engine.

5.  **Сетевой Стек (Smoltcp):**
    Сетевое взаимодействие в user-space реализовано через портированную как WASM-компонент библиотеку `smoltcp`, обеспечивающую обработку TCP/IP трафика.
