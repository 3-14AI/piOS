# TODO 2: Userland and Network

- [ ] **Пользовательское пространство почти пустое**
  В структуре репозитория есть только `userland/services` [github](https://github.com/3-14AI/piOS) — никаких реальных приложений, пакетного менеджера, libc-совместимого слоя. Запустить стороннее ПО невозможно.

- [x] **Многопользовательность — через Capabilities, но без реального теста**
  WP-048 реализует UID/GID через систему capabilities с соблюдением границ ИИ-агентами [github](https://github.com/3-14AI/piOS/blob/main/todo.md) , однако нет сценариев с несколькими реальными пользователями, сессиями, sudo-аналогами.

- [x] **Система инициализации и сервис-менеджер**
  Нет systemd-подобного init (или собственного supervisor), unit-файлов, dependency graph, graceful shutdown и watchdog.

- [x] **Система пакетов и управление приложениями**
  Нет package manager (аналог apt/pacman/cargo для WASM-компонентов), репозитория, dependency resolution и sandboxed app store.

- [x] **Полноценный installer / live-образ**
